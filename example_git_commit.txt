# Vamos ver se o arquivo que foi criado... 
run_auto ls

# Agora, vamos ver o status do git com o comando: <bold> <bg #00005f>git status</bg #00005f> </bold>
run git status$

# Observe que o arquivo <bold>exemplo.txt</bold> ainda não foi adicionado às mudanças a serem enviadas
# Para adiciona-lo, é preciso executar o comando:<bold> <bg #00005f>git add .</bg #00005f> </bold>
run git add .$

# Ótimo, vamos checar a situação atual de novo, execute:<bold> <bg #00005f>git status</bg #00005f> </bold>
run git status$

# Perceba que o arquivo exemplo.txt foi adicionado ao Index. 
# Agora ele esta presente em "Changes to be committed":
# >>> OBS: Index é apenas um lugar temporário que guarda as mudanças que você adicionar

# O próximo passo, será confirmar as mudanças feitas através do comando:<bold><bg #00005f> git commit -m "Adiciona exemplo.txt"</bg #00005f></bold>
run git commit -m "Adiciona exemplo.txt"$

# >>> OBS: Com o "git commit -m 'comentario'", o arquivo é confirmado e enviado para o HEAD, mas ainda permanece apenas no seu computador.
